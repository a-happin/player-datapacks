player.tick
==

## Features

Define following tags for registering player tick handlers

- `tag/function #player:early_tick`
- `tag/function #player:tick`
- `tag/function #player:late_tick/N`
  - requires `0 <= N <= 99`
  - The smaller `N` is, the later it will be called.
